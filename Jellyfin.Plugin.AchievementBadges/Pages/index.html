Webhook notifications (admin)

Post a message to Discord / Slack / any webhook URL whenever a user unlocks a badge. Template placeholders: {user}, {badge}, {rarity}, {description}. Discord webhook URLs (ending in /webhooks/...) work out of the box.

Inject progress (admin / debug)

Custom badges (admin)

Define your own badges tied to any existing metric. Paste a JSON array below. Each badge needs at least Id, Title, Description, Metric, and TargetValue.

Seasonal challenges (admin)

Time-limited goals. Each challenge is a badge definition with optional ChallengeStart and ChallengeEnd ISO dates. Once the window closes, earned challenges stay in the user's profile permanently.

Enable / disable badges (admin)

Disabled badges are hidden from user pages, don't count toward the completion percentage, and are auto-unequipped from users who had them equipped. Useful if your server doesn't have enough libraries or long series to ever satisfy a badge.
