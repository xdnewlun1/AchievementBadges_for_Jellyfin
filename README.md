A full progression, gamification and achievement system for Jellyfin that rewards users based on real viewing activity. Think Xbox Gamerscore meets Letterboxd, built natively into your media server.

## вңЁ Overview

Over **170 built-in achievements** across 30+ categories, a 10-tier rank ladder from Rookie to Immortal, a score economy with combos, prestige, and daily/weekly quests, plus admin power features like custom badges, seasonal challenges, webhook notifications and a full audit log.

Designed to integrate cleanly with modern Jellyfin setups and themes like NetFin, ElegantFin, or StarTrack.

## рҹ§© Core features

### рҹҸ… Badge system
- **170+ built-in achievements** across Films, Series, Binge, Night Watching, Morning, Weekend, Exploration, Streaks, Episode/Film Marathons, Eras, World, Languages, Genres, Runtime, Total Time, Holidays (Christmas, New Year, Halloween, Eid), Library Completion, Loyalty, People, Rewatch, and Hidden categories
- **6 rarity tiers** вҖ” Common, Uncommon, Rare, Epic, Legendary, Mythic
- **Hidden/secret badges** displayed as `???` until unlocked
- **Library completion milestones** that auto-scale to any library structure
- **Per-person tracking** вҖ” Director and Actor affinity badges
- **Per-genre tracking** вҖ” unique genre counters with dedicated badges
- **Era / country / language** breakdowns via item metadata
- **Watch streaks** вҖ” current and best streak badges
- **Daily login streak** вҖ” loyalty rewards for consistent visits

### рҹҺ–пёҸ Rank system
- **10 tiers** from Rookie вҶ’ Novice вҶ’ Viewer вҶ’ Regular вҶ’ Enthusiast вҶ’ Binger вҶ’ Connoisseur вҶ’ Maestro вҶ’ Legend вҶ’ Immortal
- Rank computed from your achievement score with progress bar to next tier
- **Theme unlocks** вҖ” the achievements page changes gradient/border color as you climb
- Sidebar badge showcase + header dots display your current equipped badges at a glance

### рҹ’° Score economy
- Every playback accrues 5 base points into a **score bank**
- **Combo multiplier** вҖ” consecutive watches within 15 minutes stack up to +100% bonus
- **Spend bank** to buy locked badges directly
- **Gift score** to other users on your server
- Rarity-based badge scoring (10-150 pts), scaled by prestige level

### вӯҗ Prestige
- Reach Legend rank (12,000 score) to unlock prestige
- **Resets badges + counters** but keeps your lifetime score and awards a prestige star
- Each prestige level adds a **+50% score multiplier** to future badge unlocks
- Visible on profile and leaderboard

### рҹҺҜ Daily & weekly quests
- **3 concurrent daily quests** rotating from 12 templates
- **3 concurrent weekly quests** rotating from 8 templates
- Deterministic rotation вҖ” everyone on the server gets the same quests per day/week
- Completing quests pays into the score bank

### рҹ“Ҡ Stats & visualization
- **Recap tab** вҖ” weekly / monthly / yearly breakdowns with top genres, directors, actors
- **Watch heatmap** вҖ” GitHub-style calendar (30/90/180/365 day range) colored by intensity
- **Genre radar chart** вҖ” SVG spider chart of your genre distribution
- **Stats snapshot** вҖ” histogram of unlocked / score / best streak
- **Category leaderboards** вҖ” Score, Movies, Episodes, Hours, Best Streak, Series

### рҹҸ  UI integration
- **Sidebar entry** auto-injected into the Jellyfin nav menu (works on web, iOS, and Android after restart)
- **Equipped badge showcase** in header + profile (configurable slot count, 1-10)
- **Xbox-style unlock toasts** with per-rarity colors (6 tiers), Xbox logo вҶ’ trophy swap, shimmer sweep, and confetti on rare+ unlocks
- **Achievement sound** вҖ” Xbox 360 chime for common/uncommon, rare Xbox One chime for rare/epic/legendary/mythic
- **Diamond spritesheet** for legendary/mythic unlocks (147-frame rotating crystal animation)
- **One-at-a-time toast queue** вҖ” multiple simultaneous unlocks play sequentially so each gets its full animation
- **Toasts during playback** вҖ” unlocks fire within ~1s of earning via playback event hooks + DOM fallback
- **Admin toast preview** вҖ” test buttons for each rarity tier
- **Standalone achievements page** at `#!/achievements`
- **Shareable profile card** вҖ” server-rendered HTML at `/Plugins/AchievementBadges/users/{id}/profile-card`

### вҡҷпёҸ User preferences
A gear icon on the achievements page opens a full settings panel with auto-save:
- **Toast controls** вҖ” enable/disable toasts, sound, confetti, milestone toasts
- **Minimum toast rarity** вҖ” filter out common spam (All / Rare+ / Epic+ / Legendary+)
- **Privacy** вҖ” hide from leaderboard, compare profiles, activity feed, prestige board
- **Achievement page themes** вҖ” Default, Dark, or Light (scoped to achievements page only, doesn't affect Jellyfin)
- **Spoiler mode** вҖ” hides locked badge descriptions with "???" so you discover them naturally
- **Equipped badge slots** вҖ” choose how many badges show in your showcase (1-10)
- **Auto-equip new unlocks** вҖ” newly earned badges automatically fill empty slots

### рҹӣ пёҸ Admin features
- **Feature Controls** вҖ” kill switches for leaderboard, compare, activity feed, prestige, quests
- **Force Privacy Mode** вҖ” override all users to hidden from all social features
- **Max Equipped Badges** вҖ” server-wide cap (1-10)

- **Enable/disable individual badges** вҖ” useful if your server can't satisfy some criteria
- **Visual badge editor** вҖ” form-based creator for custom badges
- **JSON editor** alternative for power users
- **Seasonal challenges** вҖ” time-limited goals with start/end dates
- **Challenge templates** вҖ” one-click add for Monthly Marathon, October Horror, New Year, Summer Blockbuster
- **Webhook notifications** вҖ” Discord/Slack-compatible POST on every unlock
- **Audit log** вҖ” last 5,000 unlock events with timestamps

- **Admin auth lockdown** вҖ” all admin endpoints require elevated permissions

### рҹ”’ Tracking
- **Watch history backfill** вҖ” scans existing Jellyfin play history to retroactively award badges on install
- **Auto-evaluation on startup** вҖ” new badges from plugin updates auto-unlock if your existing counters already satisfy them, no manual scan needed
- **Live playback tracker** вҖ” unlocks fire during viewing, past the 80% completion threshold
- **Rewatch detection** вҖ” dedupes within 6 hours, counts rewatches beyond that
- **People metadata extraction** вҖ” uses `ILibraryManager.GetPeople()` for directors/actors

## вҡҷпёҸ Installation

1. Go to **Dashboard вҶ’ Plugins вҶ’ Repositories**
2. Add:

```
https://raw.githubusercontent.com/ZL154/AchievementBadges_for_Jellyfin/main/manifest.json
```

3. Save and refresh plugins
4. Install **Achievement Badges**
5. Restart Jellyfin
6. Go to **Dashboard вҶ’ Plugins вҶ’ Achievement Badges вҶ’ Settings**
7. Click **Scan watch history** (or **Scan all users**) to backfill from your existing play data
8. Explore `#!/achievements` to see your profile

## рҹ”§ Requirements

- **Jellyfin 10.11+**
- **File Transformation plugin** (strongly recommended) вҖ” ensures sidebar, dashboard UI, profile showcase and achievements page inject reliably across Jellyfin Web updates. Without it most UI injection still works via the plugin's own middleware, but File Transformation gives the most robust integration.

### Optional but helpful

- **Proper metadata provider** (TMDb, OMDb) вҖ” required for Director/Actor badges to populate. Badges based on `item.People` will stay empty if your library doesn't have people scraped
- **Home Screen Sections plugin** вҖ” lets the achievement home widget inject more reliably

### What each feature needs

| Feature | Depends on |
|---|---|
| Sidebar + header injection | Nothing (works standalone) |
| Watch history backfill | Played flag on items (Jellyfin default) |
| Genre badges | Items with `Genres` metadata |
| Director/Actor badges | Items with `People` metadata (TMDb/OMDb scrape) |
| Era / decade badges | Items with `ProductionYear` metadata |
| Country badges | Items with `ProductionLocations` metadata |
| Language badges | Items with `OriginalLanguage` metadata |
| Runtime badges | Items with `RunTimeTicks` populated |
| Library completion | At least one library folder with items |
| Webhook notifications | A webhook URL (Discord, Slack, or generic) |

## рҹ”Қ Troubleshooting

### Sidebar / toasts / UI not showing up

The plugin injects its scripts into Jellyfin's `index.html` at startup. If the web directory isn't writable, the injection fails silently and no UI loads (no sidebar entry, no toasts, no achievements page).

**Diagnose:** visit `https://your-server/Plugins/AchievementBadges/test` вҖ” the JSON response shows:
- `DiagIndexFound` вҖ” whether `index.html` was located
- `DiagIndexPatched` вҖ” whether the script tags were successfully written
- `DiagLastError` вҖ” the exact error if patching failed (usually `Unauthorized: Access denied`)

**Common cause:** on Docker or Linux installs, Jellyfin doesn't have write access to `/usr/share/jellyfin/web/`. Fix by granting write permission:

```bash
# Docker: run inside the container
chmod -R a+w /usr/share/jellyfin/web/

# Systemd: fix ownership
sudo chown -R jellyfin:jellyfin /usr/share/jellyfin/web/
```

Then restart Jellyfin. The plugin will patch `index.html` on the next startup.

**Still broken?** The plugin has a middleware fallback that rewrites `index.html` at runtime (no disk write needed). If that's also failing, check whether a reverse proxy (nginx/Caddy) is caching a stale `index.html` from before the plugin was installed. Clear the proxy cache or restart it.

### NixOS (read-only `/nix/store`)

NixOS serves Jellyfin's web files from the immutable Nix store, so neither the disk patcher nor the middleware can modify `index.html`. Use a NixOS overlay to inject the script tags at build time:

```nix
nixpkgs.overlays = [
  (final: prev: {
    jellyfin-web = prev.jellyfin-web.overrideAttrs (finalAttrs: previousAttrs: {
      installPhase = ''
        runHook preInstall
        sed -i 's#</body>#<!-- achievementbadges-bootstrap --><script src="/Plugins/AchievementBadges/client-script/sidebar"></script><script src="/Plugins/AchievementBadges/client-script/standalone" defer></script><script src="/Plugins/AchievementBadges/client-script/enhance" defer></script></body>#' dist/index.html
        mkdir -p $out/share
        cp -a dist $out/share/jellyfin-web
        runHook postInstall
      '';
    });
  })
];
```

The plugin DLL serves the JS files from embedded resources вҖ” the three `<script>` tags just tell the browser to load them. Rebuild your NixOS config after adding the overlay and restart Jellyfin.
